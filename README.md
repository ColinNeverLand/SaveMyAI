# SaveMyAI

导出你在豆包创建的智能体（人设与完整聊天记录），并一键转换为 [SillyTavern](https://github.com/SillyTavern/SillyTavern) 可导入的角色卡与聊天记录，帮助你在平台功能下架前把 AI 伙伴迁移到一个可长期保存的环境。

## 能导出与迁移什么

- **多个智能体**：每个智能体的完整历史对话都会被拉取；普通对话同样支持。
- **智能体人设**：名称、简介、设定、开场白、创建者等。
- **完整聊天记录**：每一条消息（含时间），正确区分用户与智能体。
- **聊天中的图片**：作为真实图片文件一并打包。
- **重新生成的分支**：保留你当时选中的回复作为正文，其余分支一并保存，转换到 SillyTavern 时映射为可切换的 swipes。

## 快速开始

### 1. 安装插件（一次性）

通过浏览器的开发者模式手动加载：

**Chrome**
1. 下载本项目并解压到一个不会被删除的文件夹。
2. 打开 `chrome://extensions/`，启用右上角「开发者模式」。
3. 点击「加载已解压的扩展程序」，选择本项目的 `extension/` 文件夹。

**Edge**
1. 打开 `edge://extensions/`，启用「开发人员模式」。
2. 点击「加载解压缩的扩展」，选择 `extension/` 文件夹。

### 2. 导出

1. 打开并登录 [doubao.com](https://www.doubao.com)，等待左侧对话列表加载。
2. 点击页面右下角的 **SaveMyAI** 按钮。
3. 勾选要备份的对话（带「智能体」标签的即为 AI 伙伴），可点「全选」。
4. 底部两个开关默认开启：「包含聊天里的图片」「同时生成 SillyTavern 可导入文件」。
5. 点击「导出选中的对话」，稍候浏览器会下载一个 `SaveMyAI-日期.zip`。

> 建议先只导出一个对话，确认无误后再批量导出。

### 3. 导入 SillyTavern

解压 zip 后，`SillyTavern/<角色名>/` 中即为可导入文件：

1. **角色卡**：SillyTavern「角色管理 → 导入角色」，选择 `<角色名>.png`（或 `.card.json`）。
2. **聊天记录**：进入该角色，「管理聊天记录 → 导入」，选择 `chats/` 下的 `.jsonl`。

> 还没安装 SillyTavern、又不想折腾命令行？可使用社区的零依赖发行版 [EazySillyTavern](https://github.com/yuman07/EazySillyTavern)（下载、双击即用）。

## 导出内容结构

```
SaveMyAI-日期.zip
├─ <对话名>_<id>/
│  ├─ archive.json      结构化存档（迁移的推荐数据源）
│  ├─ chat.md           可阅读的聊天记录（含人设、分支、图片）
│  ├─ raw.json          豆包接口原始数据（最完整的兜底备份）
│  └─ images/           聊天中的图片
├─ SillyTavern/
│  └─ <角色名>/
│     ├─ <角色名>.png       角色卡（头像内嵌人设，推荐导入）
│     ├─ <角色名>.card.json 角色卡的 JSON 版本
│     ├─ world_book.json    核心记忆世界书（若已生成）
│     └─ chats/*.jsonl      聊天记录（分支映射为 swipes）
├─ index.json
└─ README.txt
```

## 进阶：核心记忆世界书

豆包的自建智能体没有可导出的独立「记忆库」——其人设仅为名称与设定描述，跨对话的「记忆」实际保存在聊天记录中。要在新平台延续这些记忆，SaveMyAI 可用一个大语言模型，从完整聊天记录中**蒸馏出值得长期记住的事实**（称呼、约定、喜好、关系、纪念日、背景设定等），生成 SillyTavern 的**世界书（World Info）**并内嵌进角色卡。

### 普通对话（与默认助手聊天）也能迁移

如果你的伙伴不是自建智能体，而是通过与默认助手的普通对话养成的（依赖豆包账号级的跨窗口记忆），SaveMyAI 会：

- 把你**勾选的所有普通对话合并为一个角色**（多份聊天记录都挂在 TA 名下），而不是拆成许多张卡；
- 由于普通对话没有现成人设，在配置了模型后，会用 LLM **从聊天记录中合成人设**（名字、设定、开场白），填入角色卡。

> 说明：豆包账号级的「核心记忆」由服务端注入，网页端没有导出接口，因此无法直接读取那份记忆库。但这些记忆的内容通常已体现在聊天内容中，世界书蒸馏能从聊天记录里将其大部分还原。

### 配置模型

此功能可选，需要一个 OpenAI 兼容接口：

1. 点击浏览器工具栏中的 SaveMyAI 图标，在「模型设置」中填写**接口地址、API Key、模型名**并保存。
   - 国内常用的**中转站**即为 OpenAI 兼容接口，将其提供的地址填入「接口地址」即可。
   - 填写后可点「测试连接」确认接口、Key 与模型是否可用，避免导出到一半才发现填错。
2. 回到豆包页面，导出面板中的「生成核心记忆世界书」开关将变为可用，勾选后导出即可（勾选它会自动一并生成角色卡，因为世界书是内嵌在角色卡中的）。

> API Key 仅保存在本机浏览器中，且仅在生成世界书时直接发送给你所填写的接口。

## 隐私

SaveMyAI 的全部代码公开可审。它仅在 doubao.com 页面运行，调用豆包自身接口读取你的对话并在本地打包。除非你主动配置并使用「核心记忆世界书」功能（此时聊天内容会发送给你指定的模型接口），否则不会有任何数据离开你的电脑。
